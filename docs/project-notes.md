# Poznámky k projektu

## Cíl

Vytvořit profesionálně působící školní web pro fiktivní politickou stranu `Strana Pražských žabáků`.

## Struktura

- `index.html` je hlavní landing page
- `kandidatka.html` je samostatná kandidátka
- `assets/css/style.css` drží celý vizuál
- `assets/js/main.js` řeší menu, animace a fake formulář

## Designové rozhodnutí

- zelená paleta podle `colors.txt`
- profesionální a serióznější tón veřejných textů
- minimum textu, důraz na layout, rytmus sekcí a připravená místa pro obrázky
- scroll-snap a reveal animace pro modernější prezentaci

## Obsahový tón

- texty vycházejí z nejnovějšího podkladu `PP - SPŽ.docx`
- hlavní stránka shrnuje program do stručných, ale věcných bloků
- kandidátka používá odborné rozdělení témat podle nového dokumentu
- footer vždy jasně říká, že jde o fiktivní školní projekt
