# Strana Pražských žabáků

Statický školní web fiktivní politické strany postavený čistě přes `HTML`, `CSS` a `JavaScript`.

## Co je hotové

- hlavní stránka `index.html`
- samostatná podstránka `kandidatka.html`
- responzivní layout pro mobil, tablet i desktop
- placeholdery pro obrázky s jasnými názvy
- prezentační kontakt formulář bez backendu
- připraveno pro jednoduché hostování přes GitHub Pages

## Lokální spuštění

V kořenové složce spusť jednoduchý server, například:

```bash
python3 -m http.server 8080 --bind 0.0.0.0
```

Pak otevři:

- `http://localhost:8080`
- nebo IP adresu počítače v LAN z telefonu

## Obsah a styl

- paleta barev vychází ze souboru `colors.txt`
- logo využívá dodaný PNG soubor
- veřejné texty vycházejí z dokumentu `PP - SPŽ.docx`
- obsah je fiktivní a není určený pro skutečnou politickou komunikaci
